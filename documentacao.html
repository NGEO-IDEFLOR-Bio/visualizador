Visualizador Geoespacial

Documentação do Aplicativo Web

Visão Geral

O Visualizador Geoespacial do IDEFLOR-Bio é uma ferramenta web interativa, projetada para explorar e analisar dados geográficos do estado do Pará. Ele oferece acesso fácil a informações importantes como limites de municípios, Unidades de Conservação, terras indígenas e massas d'água, tudo em um ambiente de mapa dinâmico e fácil de usar.

Este aplicativo foi desenvolvido com o intuito de visualizar áreas protegidas de diversas bases de dados públicas em um único portal. O objetivo é auxiliar e otimizar a visualização desses dados durante reuniões.

Funcionalidades

Seletor de Camadas

Ative ou desative diferentes camadas do mapa, como municípios, regiões de integração, UCs estaduais e federais, terras indígenas e territórios quilombolas. Este controle permite personalizar quais dados são exibidos no mapa.

Visualização e Legenda

Uma legenda dinâmica no canto inferior direito mostra as cores e símbolos das camadas visíveis. A legenda se atualiza automaticamente conforme as camadas são ativadas ou desativadas, garantindo que você sempre saiba o que está vendo.

Filtro por Atributos

A janela de filtro no canto superior esquerdo permite que os usuários selecionem uma camada, escolham uma coluna de seus atributos e apliquem um filtro com base em um valor específico. Isso é ideal para isolar e visualizar feições com características em comum para análises mais precisas.

Ferramenta de Identificação (`Identify`)

Similar a um SIG (Sistema de Informação Geográfica), a ferramenta de identificação permite ao usuário clicar em qualquer ponto do mapa para obter informações detalhadas sobre todas as camadas visíveis naquele local.

Download de Dados

Após aplicar um filtro, é possível baixar os dados filtrados no formato GeoJSON para uso em outras plataformas e softwares de geoprocessamento. Isso facilita a exportação de dados para análise offline ou para integração com outras ferramentas.

Reordenação de Camadas

O painel de controle agora inclui uma lista interativa que permite reordenar as camadas temáticas. Simplesmente clique e arraste uma camada para cima ou para baixo na lista para alterar sua ordem de visualização no mapa. Isso oferece controle total sobre qual camada aparece na frente da outra, facilitando a análise visual.

Tecnologias Utilizadas

O aplicativo foi construído utilizando uma combinação de tecnologias web modernas, otimizadas para visualização de dados geoespaciais em navegadores:

HTML, CSS & JS

Estrutura e lógica do aplicativo.

Leaflet.js

Biblioteca de mapas interativos de código aberto.

Vector Tiles & Leaflet.VectorGrid

Formato otimizado para carregamento e navegação de grandes volumes de dados de massa d'água.

GeoJSON

Formato padrão para as camadas de limite.

Tailwind CSS

Framework utilitário para o design e a estilização.
